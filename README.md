# 📝 Registration Form Project

A simple and responsive **Registration Form** built using **HTML, CSS, and JavaScript**. This project demonstrates form design, input validation, and basic user interaction.

---

## 🚀 Features

* 📋 User-friendly registration form
* ✅ Form validation using JavaScript
* 🎨 Clean and responsive UI design
* 📱 Mobile-friendly layout
* ⚠️ Error messages for invalid inputs

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

---

## 📂 Project Structure

```
Registration-Form/
│── index.html
│── style.css
│── script.js
│── README.md
```

## ⚙️ How to Use

1. Clone the repository:

```
git clone https://github.com/mukeshshan/Registration-Form.git
```

2. Open the folder:

```
cd Registration-Form
```

3. Run the project:

* Open `index.html` in your browser
